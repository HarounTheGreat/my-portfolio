# Haroun Portfolio

Personal portfolio site built with React and Vite.

## Scripts

- `npm start` runs the Vite dev server.
- `npm run build` creates a production build in `dist/`.
- `npm run deploy` publishes the `dist/` folder with `gh-pages`.

## Notes

- The app is configured to build with the `/my-portfolio/` base path for GitHub Pages deployment.
- Resume PDFs are served from the `public/` directory and selected dynamically from [`src/roleConfig.js`](/Users/macbook/Sites/personal%20projects/my-portfolio/src/roleConfig.js).
